# latynka-js
latynka-js - Javascript library for latinization of ukrainian texts

## Development

Use npm (as root) to install dependencies:

`npm install`

To test:

`make test`

## Used by
* [Latynka: Firefox Addon](https://addons.mozilla.org/en-US/firefox/addon/latynka/)
